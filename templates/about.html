About This Project

เกี่ยวกับโปรเจกต์และข้อจำกัด

ระบบประเมินความเสี่ยงภาวะผิดปกติการนอนด้วย Machine Learning โดยใช้ปัจจัยด้านสุขภาพ พฤติกรรมการนอน และไลฟ์สไตล์เป็นข้อมูลนำเข้า

Project Overview

Sleep Disorder Risk Screening

โปรเจกต์นี้พัฒนาเว็บแอปสำหรับคัดกรองความเสี่ยงภาวะผิดปกติการนอน โดยใช้โมเดล Machine Learning เพื่อจำแนกผลลัพธ์ออกเป็น 3 กลุ่ม ได้แก่ None, Insomnia และ Sleep Apnea

Dataset 374 Records Sleep Health and Lifestyle Dataset
01

เป้าหมายของโปรเจกต์

สร้างระบบคัดกรองเบื้องต้นที่ช่วยประเมินแนวโน้มความเสี่ยงจากข้อมูลสุขภาพและพฤติกรรม เพื่อแสดงผลบน Web App ที่ใช้งานง่ายและเหมาะกับการนำเสนอโปรเจกต์

02

ข้อมูลที่ใช้ในการประเมิน

  • ข้อมูลพื้นฐาน เช่น เพศ อายุ และอาชีพ
  • ข้อมูลการนอน เช่น ระยะเวลาการนอนและคุณภาพการนอน
  • ข้อมูลไลฟ์สไตล์ เช่น ความเครียด กิจกรรมทางกาย และจำนวนก้าว
  • ข้อมูลสุขภาพ เช่น BMI ความดันโลหิต และอัตราการเต้นหัวใจ
03

เทคโนโลยีที่ใช้

  • Python / Flask สำหรับ Web Backend
  • scikit-learn สำหรับ Machine Learning Model
  • HTML, CSS, JavaScript สำหรับหน้า Web App
  • Joblib สำหรับโหลดโมเดลและไฟล์ประกอบจาก Notebook
04

ข้อจำกัดของระบบ

  • ชุดข้อมูลมีขนาดจำกัด จึงเหมาะกับการศึกษาและการสาธิต
  • ผลลัพธ์เป็นการคัดกรองเบื้องต้น ไม่ใช่การวินิจฉัยทางการแพทย์
  • อาชีพที่ไม่มีใน dataset จะถูก map ไปยังกลุ่มงานที่ใกล้เคียงที่สุด
  • ควรตีความผลร่วมกับบริบทจริงและคำแนะนำจากผู้เชี่ยวชาญ
หมายเหตุสำคัญ: Web App นี้จัดทำเพื่อการเรียนรู้และการนำเสนอโปรเจกต์ Machine Learning เท่านั้น หากมีปัญหาการนอนอย่างต่อเนื่อง ควรปรึกษาแพทย์หรือผู้เชี่ยวชาญด้านการนอน
